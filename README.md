# CutConvert 🎬🎵

O **CutConvert** é uma ferramenta de mídia completa e intuitiva, desenvolvida em Flutter, para facilitar o dia a dia de quem precisa manipular áudios e vídeos rapidamente.

O projeto está em sua fase inicial e em constante evolução, com o objetivo de se tornar uma central de ferramentas de mídia potente e leve para o computador.

## Funcionalidades Principais

-   **Cortar Mídia:** Remova partes indesejadas ou extraia clipes específicos de arquivos de vídeo e áudio com precisão.
-   **Conversão de Formatos:** Mude o formato de seus arquivos (MP4, MKV, M4A, MP3, WAV, etc.) de forma simples.
-   **Compressão Inteligente:** Reduza o tamanho de vídeos para que eles se moldem aos limites de compartilhamento do **WhatsApp** e **Discord**, mantendo a compatibilidade (H.264/AAC).
-   **Fila de Processamento (Batch):** Converta dezenas de arquivos de uma só vez (ex: converter uma pasta cheia de M4A para MP3).
-   **Extração de Áudio:** Remova o vídeo e salve apenas a trilha sonora.

## 📥 Downloads e Instalação

O projeto já está buildado para Windows. Você pode baixar o executável diretamente na seção de **Releases** aqui no GitHub. 

> *Nota: Como o projeto utiliza o FFmpeg para o processamento, os binários necessários já estão inclusos no pacote para garantir que tudo funcione "out-of-the-box".*

## 🛠️ Tecnologias Utilizadas

-   [Flutter](https://flutter.dev/) - Framework UI.
-   [FFmpeg](https://ffmpeg.org/) - O motor poderoso por trás de todo o processamento de mídia.
-   [Riverpod](https://riverpod.dev/) - Gerenciamento de estado robusto.
